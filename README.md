## 🚀 Quick Start

```bash
# Clone & Setup
git clone https://github.com/yourusername/pet-adoption-app.git
cd pet-adoption-app
flutter pub get

# Run App
flutter run                    # Mobile
flutter run -d web            # Web

# Build
flutter build apk --release   # Android APK
flutter build web --release   # Web
```
